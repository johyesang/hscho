1-2. 코드 구조

Statement(문)

서로 다른 문은 세비콜론 (;) 으로 구분해야함. 단, 줄바꿈이 있으면 하나의 문으로 사용가능

즉, alert('Hello'); alert('world'); 형태나
alert('Hello')
alert('world')
와 같은 형태로 사용할 수 있는 것이다.


1-3. 변수와 상수

변수

변수를 선언할 때 let 키워드를 사용하여 let user = 'John' 식으로 표현 (var 를 사용할 수도 있지만 예전 방식임) 단 변수는 숫자로 시작 불가, '$', '_'와 같은 특수 문자만 허용

상수

변화하지 않는 값을 선언할 때는 'const'를 사용 함. 상수를 사용할 때의 상수명은 대문자와 '_'를 조합하여 표기함

const COLOR_RED = "#F00";
const COLOR_GREEN = "#0F0";
const COLOR_BLUE = "#00F";
const COLOR_ORANGE = "#FF7F00";

// 색상을 고르고 싶을 때 별칭을 사용할 수 있게 되었습니다.
let color = COLOR_ORANGE;
alert(color); // #FF7F00
